# Webpack boilerplate

Build template on webpack

# Main tools

Webpack, Babel, Scss, Pug, DevServer, autoPreFixer

## Installation

```
$ npm install
```

## Usage

### Run

```
$ npm start
```

### Build

Run build in the dev environment.
```
$ npm run build
```

Run build in the stage environment.
```
$ npm run build:stage
```

Run build in the prod environment.
```
$ npm run build:prod
```

## Testing

### Lint
```
$ npm run lint
```
